Core APIs

Built on 15 Years of Intelligence

Routing API

Routes that know India's traffic patterns, delivery hotspots, and predictive ETAs built from real journeys. Generate optimized paths with predictive ETAs and logistics-grade precision built from real journeys.

Benefits

  • Multi-stop routing
  • Multimodal transit planning (Truck, Car, Bike)
  • Precise ETAs
  • Scalable matrix dimensions

Use Cases

  • Delivery navigation
  • Fleet routing
  • Rerouting on deviation
  • Last-mile optimization

Geocoding Intelligence

Convert addresses to coordinates and back with 98%+ accuracy. The core data layer for navigation, tracking and operations.

Benefits

  • Continuous updates as delivery coverage expands
  • Address-level functional precision

Use Cases

  • Address validation
  • Address autofill at checkout
  • Order routing
  • Pin-dropped delivery locations
  • Current address detection
  • Geo-fencing checks

Autosuggest API

Predict user intent with every keystroke. The Autosuggest API provides real-time, context-aware location results, helping users find the right destination before they even finish typing.

Benefits

  • Enhanced user experience
  • Reduced zero-result errors
  • Improved data integrity with fewer typos and standardized inputs
  • Reduced form friction

Use Cases

  • Rider apps
  • Checkout forms
  • Internal ops tools

Map Tiles

Build better maps, faster. Custom styling and seamless cross-platform deployment help transform raw coordinates into interactive mapping experiences for navigation, visualization and logistics workflows.

Benefits

  • Cross-platform consistency
  • Custom/dynamic styling at runtime
  • Interactivity and event handling

Use Cases

  • Navigation base map
  • Route rendering
  • Geo-fencing
  • Logistics dashboards

Search API

Turn free-form queries into precise location results with semantic intent analysis. The API understands whether users are searching for an address or a place category, (e.g., "123 Hosur Rd" vs. "cafes in Koramangala") to deliver relevant results instantly.

Benefits

  • Semantic intent analysis
  • Intelligent proximity and biasing
  • Rich metadata delivery
  • Route-based discovery
  • Fuzzy logic and error tolerance
  • Precision type filtering
